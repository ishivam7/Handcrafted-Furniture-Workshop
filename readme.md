# Handcrafted Furniture Workshop Gallery

A responsive and user-friendly web application developed using **HTML5, CSS3, and Vanilla JavaScript** for showcasing handcrafted wooden furniture. The project provides a modern digital solution for managing furniture collections with search functionality, form validation, loading states, and accessibility features.


# Project Overview

The **Handcrafted Furniture Workshop Gallery** replaces manual paper records and spreadsheets with a clean and responsive web interface. It allows users to browse handcrafted furniture, search products, and add new furniture information while ensuring proper validation and secure input handling.


#  Features

-  Responsive Furniture Gallery
-  Real-time Furniture Search
-  Fully Responsive Design
