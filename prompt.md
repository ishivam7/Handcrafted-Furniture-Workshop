## Prompt

Create a responsive **Handcrafted Furniture Workshop Gallery** using only **HTML5, CSS3, and Vanilla JavaScript**.

The application should include:
- A modern homepage with a hero section.
- A furniture gallery displaying handcrafted products.
- A search feature with a "No data found" message.
- An "Add Furniture" form with validation and XSS input sanitization.
- A loading indicator for asynchronous operations.
- A responsive navigation bar with a hamburger menu for mobile devices.
- Semantic HTML5, ARIA labels, and keyboard accessibility.
- A clean monochromatic corporate design without using any external libraries such as Bootstrap, Tailwind, or React.
- Console analytics logging after successful form submission.

The website should be fully responsive, accessible, and easy to use on desktop, tablet, and mobile devices.
